# Comparison of Pipelined and Single-Cycle RV32I CPUs in VHDL

Development of two from-scratch RV32I soft-core CPUs in VHDL — one single-cycle and one pipelined — developed using Quartus Prime for architectural design and comparative study.

This repository contains the implementation of a RISC-V pipelined CPU and a RISC-V single-cycle CPU written in VHDL and developed in Quartus Prime Lite. The project was originally carried out as part of the Processor Architecture and Organization course during the 4th semester of my Computer Engineering degree at UNIVALI, Brazil.

The main objective of this project is to implement and study the architectural differences between pipelined and single-cycle CPU designs.

> ⚠️ Note: a detailed technical comparison report is still in preparation.  
> This repository currently provides the source code and project files required to inspect and reproduce the implementations.
